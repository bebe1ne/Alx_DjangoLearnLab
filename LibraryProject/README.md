LibraryProject

This project is a basic setup of a Django development environment intended to familiarize you with the workflow of Django projects.

## Getting Started

These instructions will you set up the project and run it on your local machine.

### Prerequisites

- Python installed on your system.
- Pip, the package installer for Python.

### Installation

1. **Install Django**

   Open your terminal and run the following command to install Django:

   ```bash
   pip install django
Create a Django Project

Initialize a new Django project named LibraryProject by executing:

django-admin startproject LibraryProject
Navigate to Project Directory

Change your directory to the LibraryProject:

cd LibraryProject
Running the Development Server
To start the development server, use the command:

python manage.py runserver
